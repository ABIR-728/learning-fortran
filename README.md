# DEBASIS SIR CLASS : Basic Python Concepts
## Files
- task 1.py — performs addition, subtraction, multiplication, division on two inputs.
- task 2.py — concatenates first and last name and prints a greeting.

## Requirements
- gfortran

## Run
- $ gfortran filename.f -o filename
- $ ./filename
- copy the programs.
- paste it on text editor.
- open the terminal of the folder where the filename.f file written in text editor is     saved.
- use gfortran to run this programs.
- use the command line written below , only change the filename according to your         preferences.
- gfortran filename.f -o filename .(press enter)
- ./filename .(press enter)
